### 计数排序

   * 计数排序类似桶排序，但桶粒度为1，并且每个桶记录的是对应数据出现的次数。

---

### 代码实现

    def count_sort(data):
        if len(data) <= 1:
            return
        # 桶的数量 = 最大值 + 1
        bucket_num = max(data) + 1
        # 建桶
        buckets = [0] * bucket_num
        # 开始计数
        for num in data:
            buckets[num] += 1
        # 排序：从桶里依次取数据，保证顺序从小到大
        sorted_index = 0
        for i in range(0, bucket_num):
            while buckets[i] > 0:
                data[sorted_index] = i
                sorted_index += 1
                buckets[i] -= 1
        return data
        
---

### 性能分析

   * 时间复杂度：O(n)
   * 空间复杂度：O(k * n)
