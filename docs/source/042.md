### 选择排序

   * 选择排序是一种基于交换的原地排序，同样将数据划分为无序区间和有序区间，从无序区间中选择最小值并将其放置于有序区间的末尾，进行 n 次重复过程来来实现整体有序。

   * 起始有序区间为空，无序区间为 a[0...n]。

---

### 代码实现

    def select_sort(data):
        length = len(data)
        if length <= 1:
            return
        for i in range(0, length):
            # 求i到length(未排序区间)的最小值
            min_index = i
            min_value = data[i]
            for j in range(i, length):
                if data[j] < min_value:
                    min_index = j
                    min_value = data[j]
            # 通过交换把最小值放到首位
            data[i], data[min_index] = data[min_index], data[i]
        return data
        
---

### 性能分析

   * 时间复杂度：选择排序进行了 n 次交换操作，单次交换操作耗费时间复杂度为 O(n)，因此整体时间复杂度为 O(n²)。
   * 空间复杂度：选择排序也是一种原地排序，没有借助额外空间，因此整体空间复杂度为 O(1)。

---

### 关键点

   1. 通过不断比较来计算未排序区间的最小值
   2. 交换时交换数组元素而不是局部变量
